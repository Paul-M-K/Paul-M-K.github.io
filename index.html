Miyoo mini plus: Game rating

Data Engineering Project with Python

I have been working with the Miyoo Mini Plus. This remarkable retro gaming hand-held device boasts an extensive library of thousands of retro games. With such a vast collection, it became challenging for me to identify the best-rated games across different consoles.

The solution I created leverages the API of IGDB (Internet Game Database). By pulling game data from IGDB and the Miyoo Mini Plus, I can match the game ROMs (Read-Only Memory files). I then create a weighted average from the ratings and the number of user ratings. I utilize this information to rename the ROMs associated with each game, ensuring that the highest-rated games surface prominently at the top for each console.

This approach not only streamlines the gaming experience but also allows me and hopefully, other users to quickly discover and access the most highly regarded games across various consoles. By implementing this system, I aim to enhance the overall user satisfaction and make the process of finding top-rated retro games on the Miyoo Mini Plus console is effortless and enjoyable.

Predicting Vancouver Rental Prices

Data Science Project With Python

The project aims to gain insights into Vancouver's rental property market by utilizing machine learning techniques to predict rental prices based on data extracted from Craigslist. The project involves scraping data from Craigslist data, and tracking rental property trends. The dataset includes attributes such as latitude, longitude, number of beds, number of baths, square footage, price, and listing names.

Analysis of the data involves plotting property locations, examining attribute correlations, and utilizing three regression models: Linear Regression, Decision Tree Regression, and RandomForestRegressor. The RandomForestRegressor model with tuned hyperparameters demonstrates the lowest Root Mean Square Error (RMSE), allowing for accurate predictions of rental prices.
